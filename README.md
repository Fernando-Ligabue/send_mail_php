# send_mail_php


App para envio de e-mails, constrída em php. 
Desenvolvida durante o curso Desenvolvimento web completo, Udemy On Line Courses.

Foram utilizados PHP 8.0 para desenvolver o back end, Xampp v3.2.4 para simular servidor Apache, Lib PHPMailer para configurações de envio de e-mail e  HTML5 e Bootstrap 4 para desenvolver as páginas front end.

Ficheiros privados, que contem dados de contas de e-mail e informações sigilosas foram realocados par fora da pasta pública (htdocs) do servidor apache, como forma de garantir que o acesso a estes ficheiros não fossem acessados por utilizadores sem autorização.
